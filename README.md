# ProductionManagerCockpit

## Technology Stack

	.NetCore->Cordova(Typescript->Angular->Ionic)

## How to Start

	 1) docker build -t productionmanagercockpit .
	 2) docker run -p 5002:5002 productionmanagercockpit
	 3) http://localhost:5002/
